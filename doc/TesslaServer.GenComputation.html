    tessla_server v0.0.1 TesslaServer.GenComputation behaviour

    Base Module to work with Modules implementing the GenComputation behaviour. Note that Computations should implement specific GenServer calls, this cant be expressed right now by callbacks because the callbacks are allready defined in GenServer. Look at the methods below to see which messages are sent to a Computation.

    Types

    id()

    Functions

    Adds a new child to the specified Computation

    send_event(id, event)

    Sends a new Event to the Computation that is registered with id to process it

    stop(id)

    Stops a Computation and unregisters its id from gproc

    subscribe_to_operands(id)

    Subscribes the Computation specified by id to all Computation it is a descendant of
